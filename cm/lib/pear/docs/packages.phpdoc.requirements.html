Requirements

Requirements --  Dependencies, time and memory settings

PHP-Extensions

  • XML-Support

  • Perl Compatible Regular Expressions - PCRE

Both extensions should be avaible in common PHP distributions.

Execution time

It is recommended to set the maximum execution time in the php.ini on a high value or disable it. 
max_execution_time = 0     ; Maximum execution time of each script, in seconds
I.e. PHPDoc needs between 5 and 10 min for creating an API documentation of the whole PEAR project depending on your computers performance.

Memory limit

It could be necessary to increase the memory limit. Especially when you are using Windows and PHP crashes while runing PHPDoc. 
memory_limit = 67108846		; Maximum amount of memory a script may consume (8MB)
