How to become a PEAR contributor and/or how to get CVS access to PEAR?

The rules are the same like the ones for getting a PHP CVS account. This means you should do one of the following things to improve your chances to get a CVS account.

  • Send patches for existing PEAR classes.

  • Create and maintain a new class for PEAR.

  • Rewrite and maintain an existing class to match the PEAR Coding Standards.

Requesting an account without having existing source code or public pre-work does not make much sense.

See also Using CVS for PHP Development.

You should announce your intention of commiting new stuff to the PEAR developers list. It's strongly recommended to subscribe to the list. See the support page for information about subscribing.
