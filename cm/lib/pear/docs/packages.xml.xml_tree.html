XML_Tree

XML_Tree --  Allow building of XML data structures.

Synopsis

require_once "XML/Tree.php";

Introduction

The XML_Tree package allows one to build XML data structures using a tree representation, without the need for an extension like DOMXML.

Example 1. Basic example

$tree  = new XML_Tree;
$root =& $tree->addRoot("root");
$foo  =& $root->addChild("foo");

header("Content-Type: text/xml");
$tree->dump();

XML_Tree::XML_Tree

void XML_Tree::XML_Tree ([string filename [, string version]])

This function is the constructor for the tree class. It takes two arguments: filename is the filename and the second parameter version defines the XML version that should be used (default is 1.0).

XML_Tree::addRoot

object XML_Tree::addRoot (string name [, string content [, array attributes]])

This function adds an new root node to the XML datastructure. The first parameter name stands for the name of the root element. The second parameter content represents the data between the opening and closing tag. The third parameter attributes can take an array of attributes for the node.

The function returns an object that represents the new root element.

XML_Tree::addChild

object XML_Tree::insertChild (mixed child [, string content [, array attributes]])

Adds a child node to current node.

XML_Tree::insertChild

object XML_Tree::insertChild (array path, integer pos, mixed child [, string content [, array attributes]])

Insert new child (child) into the tree (path, pos).

XML_Tree::removeChild

object XML_Tree::removeChild (array path, integer pos)

Removes a child (path, pos) from tree ( path, pos) and maintains namespace integrity.

XML_Tree::getTreeFromFile

object XML_Tree::getTreeFromFile ()

This function returns a tree object that represents the structure of the XML file that has been passed to the constructor .

XML_Tree::getTreeFromString

object XML_Tree::getTreeFromString (string str)

Create XML tree from given string str. This function returns a tree object that represents the structure of the given string.

XML_Tree::Clone

object XML_Tree::Clone ()

This function returns an object that represents a copy of the current XML tree.

XML_Tree::get

string XML_Tree::get ()

This function returns the text representation of the XML tree.
