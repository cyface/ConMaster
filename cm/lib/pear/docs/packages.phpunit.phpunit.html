PHPUnit

PHPUnit --  Base class for running a testsuite

PHPUnit::run

PHPUnit_TestResult PHPUnit::&run (&$suite)

Executes the given test suite

Parameter

  • &$suite - an object of PHPUnit_TestSuite

Returns

  • &PHPUnit_Result - A reference to a PHPUnit_Result object

Example

$suite  = New PHPUnit_TestSuite('Mathtest');
$result = PHPUnit::run($suite);
echo $result->toHtml();
