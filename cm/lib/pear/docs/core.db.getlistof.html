DB::getListOf() -- list internal DB info

Description

mixed getListOf (string $type)

Parameter

  • $type - type of requested info valid values for $type are db dependent, often: "databases", "users", "view", "functions"

Returns

  • mixed - the requested data or DB_Error when fail
