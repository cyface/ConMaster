Log_Observer

Log_Observer -- Log observer class

Synopsis

require_once "Log/Log_Observer.php";

Description

The Log_Observer class implements the Observer end of a Subject-Observer pattern for watching log activity and taking actions when notable events occur.

Log_Observer::factory

object Log_Observer::factory (string observer_type [, integer priority])

Attempts to return a concrete Log_Observer instance of the specified observer_type.
