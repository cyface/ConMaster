DB::getCol()

DB::getCol() -- Fetch a single column from a query

Description

mixed &getCol (string $query [, mixed $col [, array $params]])

Fetch a single column from a result set and return it as an indexed array.

Parameter

  • $query - the SQL query

  • $col - which column to return (integer [column number, starting at 0] or string [column name]), default is 0

  • $params - if supplied, prepare/execute will be used with this array as execute parameters

Returns

  • mixed - the first row of results as an array indexed from 0 or a DB_Error when fail
