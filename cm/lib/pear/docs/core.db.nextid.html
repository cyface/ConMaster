DB::nextId()

DB::nextId() -- returns the next free id of a sequence

Description

mixed nextId (string $seq_name, boolean $on_demand)

Parameter

  • $seq_name - name of the sequence

  • $ondemand - when true the sequence is automatic created, if it not exists. Default ist true

Returns

  • mixed - a free id or DB_Error when fail
