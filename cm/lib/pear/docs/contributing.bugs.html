Reporting bugs

If you think that you have found a bug in a PEAR package, please take care that you are using the latest version of the package and that your system does meet the packages' requirements.

If the bug still persists with the latest version of the package, don't hesitate to fill out a bug report on bugs.php.net. Please choose "PEAR related" as the category for the report. The developers on the PEAR development list will be noticed about your report.

More information and tips on how to report bugs in a proper way can be found here.

If you have a fix for the bug available, please read the next section to get to know how to submit it.
